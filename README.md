
## Краткое описание задач

1. Основной функционал задачи №1 описан в `scripts/ozon_export.py`. Результат будет сохранен в `reports/`.
2. Telegram-бот (задача №2) описан в `bot/`. Задачи №1 и №2 логически объединены и имеют общую файловую структуру, задача №3 обособлена и расположена в своей директории.
3. Задача №3 описана в отдельной директории `cleaner/`. Там же находятся исходный файл, скрипт и полученный файл.

Реализовал таким образом, что запуск скрипта №1 тянет за собой уведомление о создании сводки в ТГ, при этом в самом боте также есть кнопка для формирования сводки.

Все команды для запуска скриптов в разделе "Запуск".

## Настройка

Создайте виртуальное окружение и установите зависимости:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Заполните `.env` по примеру `.env.example`. 

## Запуск

Экспорт и отправка сводки (если заданы Telegram-настройки):

```powershell
python -m scripts.main
```

Запуск бота локально:

```powershell
python -m bot.main
```

Запуск скрипта очистки каталога:
```powershell
cd cleaner
python catalog_cleaner.py
```

## Docker

```powershell
docker compose up --build -d
```

Контейнер получает секреты из `.env`. Для планового утреннего запуска можно подтянуть любой планировщик.
